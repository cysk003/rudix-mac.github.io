a2ps: GNU a2ps

GNU a2ps is an Any to PostScript filter. Of course it processes plain text files, but also pretty prints quite a few popular languages

  • Site: http://www.gnu.org/software/a2ps/
  • Port: https://github.com/rudix-mac/rudix/tree/2015/Ports/a2ps
  • License: GPL

Install

sudo rudix install a2ps
