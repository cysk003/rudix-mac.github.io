ne: Nice Editor

ne is a free text editor based on the POSIX standard that runs on almost any UNIX machine

  • Site: http://ne.di.unimi.it
  • Port: https://github.com/rudix-mac/rudix/tree/2015/Ports/ne
  • License: GPL

Install

sudo rudix install ne
