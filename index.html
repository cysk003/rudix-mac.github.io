Welcome to Rudix!

Rudix is a collection of pre-built Unix software delivered as packages for OS X: “The hassle-free way to get Unix programs on OS X”.

Reasons to use Rudix:

  • Ready to use, don't need to compile anything.
  • Compatible with El Capitan, Yosemite, Mavericks, Mountain Lion, Lion and Snow Leopard (PowerPC is not supported).
  • Included dependencies when applicable.
  • A wide selection of packages.
  • Distribute Python modules.
  • Developer friendly.

Quick Start

There is two ways to have the Rudix Package manager installed. You can download the proper package and install it from packages > rudix or you can bootstrap by using the command line.

Just open the Terminal.app and type:

curl -O https://raw.githubusercontent.com/rudix-mac/rpm/2015.10.20/rudix.py
sudo python rudix.py install rudix

or even compressed in one-line:

curl -s https://raw.githubusercontent.com/rudix-mac/rpm/2015.10.20/rudix.py | sudo python - install rudix
  • You will be probably asked for administrator password when installing packages.
  • Consider the security risk of using the command line approach.
  • If you are having trouble to upgrade Rudix from the previous versions, please try the command line boostrap.

Now you are ready to use Rudix packages, type the command rudix search to get a list of packages for installation. For example, use sudo rudix install wget to install wget (GNU Wget). Learn more »
