paramiko: Paramiko (Python SSH module)

Paramiko is a module for python 2.2 that implements the SSH2 protocol for secure connections to remote machines

  • Site: http://www.lag.net/paramiko/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/paramiko
  • License: LGPL

Install

sudo rudix install paramiko
